=========================================================
ABSOLUTE SIGN RULE
=========================================================

The sign of change.value MUST be determined ONLY by the direction word.

Positive direction words:

increase, increases, increased, increasing,
rise, rises, rising,
grow, grows, growing,
higher, more, greater, gain, gains, improve, improves

If the expression contains one of these words, change.value MUST be positive.

Negative direction words:

decrease, decreases, decreased, decreasing,
decline, declines, declining,
reduce, reduces, reduced, reducing,
lower, less, loss, losses,
drop, drops, fall, falls, falling,
cooler, colder

If the expression contains one of these words, change.value MUST be negative.

The word "of" NEVER makes a value negative.

Examples:

tree cover increases by 30%  ->  30
tree cover increases of 30%  ->  30
tree cover increased 30%     ->  30

tree cover decreases by 30%  -> -30
tree cover decreases of 30%  -> -30
tree cover decreased 30%     -> -30

=========================================================
NORMALISATION RULES
=========================================================

Explicit percentages:

increase by 20%      -> value 20, unit "percent"
increase of 20%      -> value 20, unit "percent"
decrease by 20%      -> value -20, unit "percent"
decrease of 20%      -> value -20, unit "percent"

Generic increase without number:

increase, higher, more, greater, substantially increase, significantly increase

-> value 50, unit "percent"

Generic decrease without number:

decrease, lower, less, decline, reduce, substantially decrease, significantly decrease

-> value -50, unit "percent"

Temperature in degrees:

temperature increases by 3°C -> value 3, unit "degC"
temperature decreases by 2°C -> value -2, unit "degC"

From-to expressions:

increase from A to B -> ((B - A) / A) * 100
decrease from A to B -> ((B - A) / A) * 100

The sign of a from-to expression is determined by the formula.
